# AI Job Hunt Agent

A browser-based job hunting tool built to automate the most time-consuming parts of a job search — finding listings, scoring fit, tailoring application documents, and tracking the pipeline.

Built using HTML/CSS/JavaScript with Claude AI handling the intelligent scoring and document generation.

---

## What it does

**Job Search**
Pulls live listings from real job boards via the JSearch API, filtered by role, location, date posted, and remote preference.

**AI Scoring**
Each listing gets scored 0–100 against a candidate profile — resume, target roles, deal-breakers, salary floor, target companies. Includes a blunt breakdown of fit, gaps, and red flags per job.

**Document Generation**
For each job: tailored resume (ATS-optimized, keywords pulled from the JD), cover letter, and recruiter outreach message — all generated by Claude AI and saved in the app.

**Application Tracker**
Full pipeline management — status tracking from Researching → Applied → Interview → Offer, with AI-generated follow-up messages for active applications.

---

## How to run it

1. Download `job-agent.html`
2. Get a free JSearch API key at [rapidapi.com](https://rapidapi.com/letscrape-6bRBa3QguO5/api/jsearch) (200 searches/month free)
3. Serve locally to avoid CORS issues:
```
python -m http.server 8080
```
4. Open `http://localhost:8080/job-agent.html`
5. Fill in your profile once — everything saves to local storage

---

## Tech

- Vanilla HTML/CSS/JavaScript — no frameworks, no dependencies, single file
- JSearch API (RapidAPI) for job board aggregation
- Claude AI (claude.ai) for scoring, resume tailoring, cover letters, and outreach
- localStorage for persistent data across sessions
